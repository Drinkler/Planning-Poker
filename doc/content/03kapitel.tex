\chapter{Versionsverwaltung}
Zur Versionsverwaltung wurde ein \href{https://github.com/Drinkler/Planning-Poker}{Github Repository} eingerichtet damit mehrere Entwickler an dem Projekt arbeiten können.\\
Das Projekt wurde von jedem Entwickler geklont um lokal entwickeln zu können.\\
In dem Github Repository ist das Projekt, die Dokumentation sowie verschiedene Konfigurationsdateien zufinden.\\
